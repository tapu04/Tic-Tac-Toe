Multiplayer Tic-Tac-Toe Game
Developed a real-time multiplayer Tic-Tac-Toe game using React and Node.js.
Implemented WebSockets for seamless player communication and optimized
state management.
Designed an intuitive UI with React and managed game logic using Express.js on
the backend.
Improved game response time by 30% by optimizing server-client interactions.
Technologies Used: React.js, Node.js, Express.js, WebSockets, HTML, CSS,
JavaScript
